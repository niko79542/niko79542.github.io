Variable Notation

Let's define some notation to use throughout the rest of this blog. These come from Elements of Statistical Learning by Jerome H. Friedman, Robert Tibshirani, and Trevor H. $X$: one generic input feature $X_{j}$: the generic jth feature of input X $Y$: quantitative output $G$: qualitative output capital letter: generic variable lowercase letter: observed values
